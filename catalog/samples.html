Sample Applications

Applications demonstrating the APIs and features of Eclipse Dirigible


The sample applications show specific aspects of the usage of Eclipse Dirigible and the Enterprise JavaScript API. There are two types of samples in this category:

  • Simple Samples - demo a single feature or a single API usage. They are created to be highly reusable code snipets.
  • Solution Samples - showing more complex use cases that follow real world scenarious.
