Version [10.0.0](https://github.com/eclipse-dirigible/dirigible/releases/tag/v10.0.0) is available.

#### Themes

* Upgrade to Spring Boot 3.2
* `@dirigible` module scope renamed to `sdk`
* Phase-out of CommonJS `require` in favour of ES6 `import`
* TypeScript-based application templates
* Job, listener and integration handlers move to ES6 / TypeScript

* Source tag: [v10.0.0](https://github.com/eclipse-dirigible/dirigible/tree/v10.0.0)
* Milestone (bug-fixes and enhancements): [#53](https://github.com/eclipse-dirigible/dirigible/milestone/53?closed=1)
* Maven Central: [org.eclipse.dirigible](https://search.maven.org/search?q=org.eclipse.dirigible)
