# SDK references

Eclipse Dirigible ships two first-class SDKs:

- **TypeScript / JavaScript** -> [`@aerokit/sdk/*`](/api/)
- **Java** -> [`org.eclipse.dirigible.sdk.*`](/sdk/)

Both share the same in-process platform services, so a Java listener and a TypeScript producer (or vice versa) interoperate transparently.
