# Get Started

The fastest path to a running Dirigible.

## Pick an installation method

- **[Docker](/help/get-started/installation/docker)** — the recommended way to try Dirigible locally.
- **[Standalone JAR](/help/get-started/installation/jar)** — self-hosted on any machine with a JDK.
- **[Kubernetes](/help/get-started/installation/kubernetes)** — production-style deployment.
- **[Trial](/help/get-started/installation/trial)** — hosted environment, no install.
