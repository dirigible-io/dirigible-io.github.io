# Modelers

Modelers produce model artefacts (`*.edm`, `*.dsm`, `*.bpmn`, `*.form`, `*.camel`) that are either reconciled directly by a synchronizer or used as input to template-based generation.

- **[Entity Data Modeler](/help/ide/modelers/entity-data)** — `*.edm` / `*.model`. Generates entities + CRUD UIs + OData services.
- **[Database Schema Modeler](/help/ide/modelers/database-schema)** — `*.dsm`. Generates `.schema` / `.table` / `.view` artefacts.
- **[BPMN Modeler](/help/ide/modelers/bpmn)** — `*.bpmn`. Flowable process definitions.
- **[Form Designer](/help/ide/modelers/form-designer)** — `*.form`. HTML form layout.
- **[Integrations (Karavan)](/help/ide/modelers/integrations-karavan)** — `*.camel`. Apache Camel route designer.
