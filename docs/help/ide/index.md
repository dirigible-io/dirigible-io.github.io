# In-Browser IDE

Dirigible's primary entry point is the browser IDE at `http://localhost:8080`. It composes **perspectives** (workspaces for a task), **editors** (per-artefact authoring), **modelers** (visual designers), and **views** (side panels) - all served by the same JVM that runs your application code.

## Sections

- **[Shell and branding](/help/ide/shell-and-branding)** - top-level frame, themes, branding hooks.
- **[Perspectives](/help/ide/perspectives/)** - Workbench, Database, Git, Documents, Jobs, Operations, Processes, Messaging, Tracing, Security, Settings.
- **[Editors](/help/ide/editors/)** - Monaco, CSV, access/roles, jobs, listeners, extensions, websockets, form builder, image, mapping, report.
- **[Modelers](/help/ide/modelers/)** - Entity Data, Database Schema, BPMN, Form Designer, Integrations (Karavan).
- **[Views](/help/ide/views/)** - Projects, Console, Terminal, Search, Problems, Logs, Debuggers (JS + Java), Jobs, Listeners, Extensions, Websockets, Registry, Repository, Transfer, Swagger, Preview, Properties.
- **[Menus](/help/ide/menus)** - per-perspective top bars.
- **[Keyboard shortcuts](/help/ide/keyboard-shortcuts)**.
- **[Themes](/help/ide/themes)** - BlimpKit, Classic, Mystic, High-Contrast.
