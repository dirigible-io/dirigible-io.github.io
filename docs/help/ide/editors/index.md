# Editors

Each artefact type has a dedicated editor - most of them visual, some code-based.

- **[Monaco](/help/ide/editors/monaco)** - JavaScript / TypeScript / Java / Python / HTML / CSS / JSON.
- **[CSV and CSVIM](/help/ide/editors/csv-and-csvim)** - table data and import models.
- **[Access and roles](/help/ide/editors/access-and-roles)** - URL access rules and role declarations.
- **[Jobs](/help/ide/editors/jobs)** - scheduled job definitions.
- **[Listeners](/help/ide/editors/listeners)** - message queue/topic listeners.
- **[Extensions](/help/ide/editors/extensions)** - extension points and providers.
- **[Websockets](/help/ide/editors/websockets)** - WebSocket endpoint bindings.
- **[Form builder](/help/ide/editors/form-builder)** - visual form layout.
- **[Image](/help/ide/editors/image)** - raster image preview / metadata.
- **[Mapping](/help/ide/editors/mapping)** - data mapping definitions.
- **[Report](/help/ide/editors/report)** - report templates.
