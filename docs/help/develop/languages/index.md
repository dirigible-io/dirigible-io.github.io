# Languages

Every language listed here runs in the same JVM, against the same data sources, the same broker, the same security context.

- **[JavaScript](/help/develop/languages/javascript)** — ES6+, synchronous programming model, CommonJS + ESM.
- **[TypeScript](/help/develop/languages/typescript)** — transpiled at the platform; project-root `tsconfig`.
- **[Java](/help/develop/languages/java)** — client `.java` compiled in-process by `engine-java`.
- **[Python](/help/develop/languages/python)** — server-side modules via `engine-python`.
