# Credits

Eclipse Dirigible builds on a long list of open-source projects. Among them:

## Runtime

- [Spring Boot](https://spring.io/projects/spring-boot)
- [Spring Framework](https://spring.io/projects/spring-framework)
- [Hibernate](https://hibernate.org/)
- [GraalVM](https://www.graalvm.org/) (Graalium runtime)
- [Apache Camel](https://camel.apache.org/)
- [Flowable](https://www.flowable.com/) - BPMN engine
- [Apache Quartz](https://www.quartz-scheduler.org/)
- [Apache ActiveMQ](https://activemq.apache.org/)
- [Apache CXF](https://cxf.apache.org/) - OData / SOAP
- [Apache Chemistry](https://chemistry.apache.org/) - CMIS
- [Apache Lucene](https://lucene.apache.org/)

## Frontend

- [Monaco Editor](https://microsoft.github.io/monaco-editor/)
- [AngularJS](https://angularjs.org/)
- [BlimpKit](https://blimpkit.dev/) - on top of SAP Fundamental Styles
- [mxGraph](https://github.com/jgraph/mxgraph)
- [Karavan](https://github.com/apache/camel-karavan) - Camel route designer
- [bpmn-visualization-js](https://github.com/process-analytics/bpmn-visualization-js)
- [Xterm.js](https://xtermjs.org/)
- [AG Grid](https://www.ag-grid.com/)
- [Chart.js](https://www.chartjs.org/)
- [jsTree](https://www.jstree.com/)

## Build and test

- [Maven](https://maven.apache.org/)
- [Node.js](https://nodejs.org/) and [esbuild](https://esbuild.github.io/)
- [JUnit](https://junit.org/) and [Selenide](https://selenide.org/)
