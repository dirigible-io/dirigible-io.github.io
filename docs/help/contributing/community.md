## Reporting a bug

File an issue on the main [GitHub repository](https://github.com/eclipse/dirigible/issues). Include:

- Platform version (visible in Spring Boot Admin under `/spring-admin/`).
- Database backend.
- Steps to reproduce.
- Relevant log output.

## Asking a question

Use the [GitHub Discussions](https://github.com/eclipse/dirigible/discussions) tab on the platform repository, or Slack for real-time chat.
