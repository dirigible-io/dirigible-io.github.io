# Confluence - `*.confluence`

Confluence-flavoured wiki text rendered by `engine-wiki`. Synchronizer: `ConfluenceSynchronizer`. Served alongside Markdown at `/services/wiki/<project>/<path>.html`.

## When to use it

- Migrating Confluence content into the platform.
- Authors who already think in Confluence markup.

For new content prefer Markdown unless you have an explicit reason - the toolchain support (preview, formatting, third-party editors) is wider.
