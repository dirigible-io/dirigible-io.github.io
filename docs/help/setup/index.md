# Setup

Run Dirigible yourself, from a laptop to a production cluster.

## Quick paths

- **[Docker](/help/setup/docker)** — recommended for evaluation.
- **[Standalone JAR (Tomcat)](/help/setup/tomcat)** — runs anywhere with a JDK.
- **[Kubernetes](/help/setup/kubernetes/)** — production deployment with Helm.
- **[Cloud Foundry](/help/setup/cloud-foundry)**.
- **[GraalVM native image](/help/setup/native-image)** — startup-optimised build.

## Wiring

- **[Databases](/help/setup/databases/)** — H2, PostgreSQL, MSSQL, MariaDB, MySQL, HANA, Snowflake, MongoDB.
- **[Authentication](/help/setup/authentication/)** — Basic, GitHub OAuth, Keycloak, Cognito, Snowflake.
- **[Multi-tenancy](/help/setup/multi-tenancy)** — subdomain routing, tenant provisioning.
- **[Environment variables](/help/setup/environment-variables)** — all `DIRIGIBLE_*` tunables.
